# Columbia Quant LLM Reading Group Website

Pure HTML/CSS static website for GitHub Pages.

No React. No Vite. No npm. No build step.

## Files

- `index.html` — homepage
- `schedule.html` — weekly schedule
- `notes.html` — course notes and external readings
- `people.html` — consultants, contributors, and community background
- `projects.html` — project directions and contribution model
- `policies.html` — academic discussion norms
- `style.css` — shared CSS

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to `Settings -> Pages`.
4. Choose `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save.
